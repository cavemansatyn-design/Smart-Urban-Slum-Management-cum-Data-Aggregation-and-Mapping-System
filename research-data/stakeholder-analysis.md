## Stakeholder Profiles

### Slum Dwellers

| Attribute | Description |
|-----------|-------------|
| **Goals** | Access welfare, improve living conditions, security of tenure |
| **Pain Points** | Lack of documentation, fear of eviction, low digital literacy |
| **Tech Context** | Smartphone users, often shared devices, intermittent connectivity |
| **Key Needs** | Simple UI, welfare visibility, trust in data use |

### Nearby Citizens

| Attribute | Description |
|-----------|-------------|
| **Goals** | Report encroachments, contribute to city improvement |
| **Pain Points** | Unclear impact, lengthy processes, privacy concerns |
| **Tech Context** | Smartphone and web users, varying literacy |
| **Key Needs** | Quick reporting, confirmation, optional anonymity |

### Nodal Officers

| Attribute | Description |
|-----------|-------------|
| **Goals** | Verify reports, manage cases, meet targets |
| **Pain Points** | High workload, field constraints, manual processes |
| **Tech Context** | Mobile in field, desktop in office |
| **Key Needs** | Prioritization, mobile verification, batch actions |

### Government Authorities

| Attribute | Description |
|-----------|-------------|
| **Goals** | Data-driven planning, policy decisions, accountability |
| **Pain Points** | Fragmented data, delayed reports, manual aggregation |
| **Tech Context** | Desktop, presentations, reports |
| **Key Needs** | Dashboards, exports, trends, regional views |

## Power-Interest Matrix

| Stakeholder | Power | Interest | Strategy |
|-------------|-------|----------|----------|
| Government | High | High | Key decision-maker, closely engage |
| Nodal Officers | Medium | High | Primary users, co-design |
| Slum Dwellers | Low | High | Empower through design |
| Citizens | Low | Medium | Lower friction, encourage participation |

---

## Success Criteria by Stakeholder

| Stakeholder | Success = |
|-------------|-----------|
| Slum Dweller | Can complete survey and see welfare schemes in < 5 min |
| Citizen | Can report with photo + location in < 2 min |
| Nodal Officer | Can verify 10+ cases per day with mobile support |
| Government | Can export report for meeting in 1 click |
