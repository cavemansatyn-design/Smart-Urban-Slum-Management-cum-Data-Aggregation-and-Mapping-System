# NIVAAS Research Data

## Overview

This folder contains research documentation from the HCI (Human-Computer Interaction) phase of the NIVAAS project—user studies, stakeholder analysis, and design insights that informed the prototype.

---

## Contents

| File | Description |
|------|-------------|
| [research-methodology.md](research-methodology.md) | Approach, methods, participants |
| [findings.md](findings.md) | Key insights and recommendations |
| [stakeholder-analysis.md](stakeholder-analysis.md) | Stakeholder profiles and needs |

---

## Research Goals

1. Understand barriers to reporting and participation
2. Identify usability requirements for diverse user groups
3. Validate information architecture and workflows
4. Inform design decisions for low-literacy and low-connectivity contexts

---

## Data Privacy

- All user research data is anonymized
- No personally identifiable information (PII) is stored
- Aggregated insights only in documentation
