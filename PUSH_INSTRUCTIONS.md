# Ready to Push — NIVAAS Repository

## Pre-push Checklist

- [ ] Add screenshots to `assets/images/ui-screens/` (optional — Figma link works without them)
- [ ] Add demo video to `assets/videos/demo.mp4` and `demo.gif` (if you have one)
- [ ] Add diagram images from PDF to `assets/images/` (use-case, task flows, mind map, personas)
- [ ] Rename folder to `prototype-project` or `nivaas` (optional — close Cursor first)

## Git Commands

```bash
# Initialize (if not already)
git init

# Add all files
git add .

# First commit
git commit -m "Initial commit: NIVAAS Smart Urban Settlement Mapping System"

# Add remote (replace with your GitHub repo URL)
git remote add origin https://github.com/YOUR_USERNAME/nivaas.git

# Push to main
git branch -M main
git push -u origin main
```

## Repository Name Suggestion

`nivaas` or `nivaas-prototype`
