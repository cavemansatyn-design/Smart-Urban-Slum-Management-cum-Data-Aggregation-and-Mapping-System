# NIVAAS Prototype

## Overview

This folder contains documentation and assets related to the NIVAAS HCI prototype—the initial proof-of-concept that demonstrates core user flows and interfaces.

---

## Prototype Scope

The prototype covers:

1. **Slum Dweller Interface** — Survey submission, welfare access, emergency
2. **Citizen Reporting Interface** — Report with photo and location
3. **Nodal Officer Dashboard** — Case management, verification
4. **Government Overview Dashboard** — Analytics, heatmaps, reports
