## Design Principles

1. **Accessibility** — High contrast, large touch targets, screen reader support
2. **Simplicity** — Icon-based navigation for low-literacy users
3. **Trust** — Clear feedback, transparent data usage
4. **Mobile-first** — Optimized for field use on smartphones
