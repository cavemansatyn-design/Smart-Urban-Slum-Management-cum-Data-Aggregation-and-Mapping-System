# NIVAAS UI Mockups

## Screen Overview

### 1. Slum Dweller Interface

| Screen | Description | Key Elements |
|--------|-------------|--------------|
| **Home** | Dashboard with quick actions | Survey, Welfare, Emergency buttons |
| **Survey Form** | Multi-step demographic survey | Progress indicator, simple inputs |
| **Welfare Schemes** | List of eligible schemes | Cards with scheme name, eligibility, apply link |
| **Emergency** | Emergency alert screen | Large red button, confirmation dialog |

### 2. Citizen Reporting Interface

| Screen | Description | Key Elements |
|--------|-------------|--------------|
| **Report** | Main reporting flow | Camera, location picker, description field |
| **Location Picker** | Map to confirm/adjust location | Draggable pin, address display |
| **Confirmation** | Post-submission | Reference ID, status message |

### 3. Nodal Officer Dashboard

| Screen | Description | Key Elements |
|--------|-------------|--------------|
| **Case Queue** | List of assigned cases | Filters, status badges, priority indicators |
| **Case Detail** | Single case view | Photos, location map, notes, action buttons |
| **Heatmap View** | Map with settlement density | Toggle layers, cluster click to expand |

### 4. Government Overview Dashboard

| Screen | Description | Key Elements |
|--------|-------------|--------------|
| **Analytics** | Summary metrics and charts | KPI cards, trend graphs |
| **Reports** | Filterable report list | Export, filters, pagination |
| **Heatmap** | City-wide settlement view | Region selector, time range |
