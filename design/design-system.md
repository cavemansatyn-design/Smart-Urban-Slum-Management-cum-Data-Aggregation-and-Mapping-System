# NIVAAS Design System

## Overview

The NIVAAS design system ensures consistency across all user interfaces—Slum Dweller, Citizen, Nodal Officer, and Government dashboards.

---

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary | Blue | `#2563EB` | CTAs, links, primary actions |
| Success | Green | `#059669` | Confirmations, verified states |
| Warning | Amber | `#D97706` | Pending, attention needed |
| Error | Red | `#DC2626` | Errors, emergency |
| Neutral | Gray | `#6B7280` | Body text, borders |

### Backgrounds

- **Light**: `#FFFFFF`, `#F9FAFB`
- **Dark** (optional): `#111827`, `#1F2937`

---

## Components

### Buttons

- **Primary**: Solid background, white text
- **Secondary**: Outline, primary color border
- **Emergency**: Red, prominent placement

### Cards

- White background, subtle shadow
- 8px border radius
- 16px padding

### Forms

- Clear labels above inputs
- Error states in red
- Success checkmarks for completed steps

### Icons

- Consistent 24x24px for navigation
- 48x48px for primary actions (e.g., Report, Emergency)
- Use icon-first design for low-literacy users

---

## Spacing

- Base unit: 4px
- Common: 8px, 16px, 24px, 32px
- Section gaps: 48px

---

## Responsive Breakpoints

| Breakpoint | Width | Target |
|------------|-------|--------|
| Mobile | < 640px | Slum dweller, citizen |
| Tablet | 640px – 1024px | Nodal officer (field) |
| Desktop | > 1024px | Government dashboard |
