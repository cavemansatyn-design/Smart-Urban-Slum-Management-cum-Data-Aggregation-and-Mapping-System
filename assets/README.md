## Adding Assets

1. **Cover**: Use `cover.svg` (included) or add `cover.png` (1200×400px)
2. **Diagrams**: Export from PDF or Figma at 2× resolution
3. **Screenshots**: Capture from [Figma prototype](https://www.figma.com/design/CqKJ35W9IJGI1nEKeTRJRG/MAIN?node-id=0-1&p=f)
4. **Video**: MP4 (H.264) under 50MB for GitHub; GIF under 10MB
